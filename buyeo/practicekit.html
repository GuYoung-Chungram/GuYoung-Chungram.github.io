GUIDE

프롬프트 활용 가이드

역할 (Role)

"너는 15년 경력 공공기관 홍보 전문가야"처럼 AI에게 정체성을 부여합니다. 공공 문체·용어·형식이 자동으로 반영됩니다.

맥락 (Context)

"수강생 150명 대상, 5월 19일 개최, 공식 보도자료 톤으로"처럼 대상·목적·배경·제약조건을 함께 제공하면 재작업이 줄어듭니다.

형식 (Format)

"500자 이내, 소제목 3개, 공문서 말투, 마지막에 문의처 포함"처럼 출력 결과를 정확히 지정할수록 바로 쓸 수 있는 결과물이 나옵니다.

⚠️ 반드시 기억하세요 — AI 활용 3가지 리스크

🔴 할루시네이션
AI는 틀린 정보를 자신 있게 생성합니다. 수치·법령명·사실은 반드시 공식 자료로 검증하세요.
🟡 보안
개인정보·내부 비공개 문건은 AI에 입력하지 마세요. 행정 비공개 자료의 AI 학습을 차단해야 합니다.
🔵 저작권
AI 생성물의 저작권 귀속이 불명확합니다. 공공 배포 전 저작권 검토가 필요합니다.

SECTOR 01

공문서 · 보고서 작성

기안문, 업무보고서, 결과보고서 등 일상적인 행정 문서 초안을 빠르게 생성합니다.
프롬프트 1-1
기안문 초안 작성

특정사업이나 행정 조치를 추진하기 위해 기안문을 작성해야 할 때

너는 15년 경력의 지방자치단체 공문서 작성 전문가야. 아래 정보를 바탕으로 행정 기안문 초안을 작성해줘.

[사업명] {사업명}
[추진 배경] {추진 배경 또는 목적 2~3줄}
[주요 내용] {세부 내용 및 일정}
[예산] {예산액 및 출처}
[담당부서] {부서명}

[작성 조건]
- 형식: 제목 / 추진 배경 / 추진 내용 / 기대 효과 / 향후 계획
- 어조: 공식적·간결하게, 수동태 금지
- 분량: A4 1매 이내
변수 안내{사업명} 사업 이름 / {추진 배경} 목적·필요성 / {세부 내용} 구체적 시행 내용 / {예산액} 금액·재원
추천 도구Claude (공문 형식 정확도 높음)
활용 팁기안문 작성 후 "기안문 형식에 맞게 다시 정리해줘"로 추가 보완을 요청하세요.
프롬프트 1-2
업무 보고서 작성

월간·분기 업무 실적을 상급자에게 보고할 문서가 필요할 때

너는 공공행정기관 업무보고서 전문 작성자야. 아래 내용을 바탕으로 간결하고 명확한 업무 보고서를 작성해줘.

[보고 기간] {기간 예: 2026년 5월}
[부서명] {부서명}
[주요 추진 실적] {완료된 업무 목록}
[진행 중 사항] {현재 진행 업무 및 진행률}
[주요 이슈 및 건의사항] {문제점 및 건의}

[조건]
- 핵심 메시지가 30초 내에 파악되도록 구성
- 실적은 수치로 표현 (건수, 비율 등)
- 500자 이내, 표 형식 포함
변수 안내{기간} 보고 대상 기간 / {부서명} 해당 부서 / {완료 업무} 항목별 실적 / {이슈} 문제점·건의
추천 도구Claude / ChatGPT
활용 팁수치가 없을 경우 "~건 처리", "~% 달성" 형태로 직접 채워 넣고 검증 필수.
프롬프트 1-3
사업 결과보고서 작성

행사·사업 종료 후 성과를 정리해 보고해야 할 때

공공기관 사업 결과보고서 전문가로서 아래 정보를 바탕으로 결과보고서를 작성해줘.

[사업명] {사업명}
[기간 및 장소] {기간} / {장소}
[참여 인원] {인원 수 및 대상}
[주요 성과] {성과 항목 3~5가지}
[예산 집행] {예산액 / 집행액 / 잔액}
[향후 계획] {후속 조치 또는 개선 방향}

[형식]
- 구성: 사업 개요 / 추진 경과 / 주요 성과 / 예산 집행 현황 / 향후 계획
- 어조: 객관적·공식적
- A4 1~2매 이내
변수 안내{사업명} / {기간·장소} / {인원} / {성과} / {예산} 항목 직접 입력
추천 도구Claude / Gemini
활용 팁사진·자료가 있다면 "별첨 자료를 인용한 보고서 요약" 형태로 추가 활용 가능.
프롬프트 1-4
공문 회신 초안 작성

타기관이나 민원인으로부터 받은 공문에 대한 회신이 필요할 때

행정 공문서 작성 전문가로서 아래 상황에 맞는 공문 회신 초안을 작성해줘.

[수신처] {수신 기관명 또는 민원인}
[원문 요청 내용 요약] {상대방이 요청한 내용}
[우리 기관 답변 방향] {긍정/부분 긍정/불가 및 이유}
[관련 법령 또는 근거] {관련 법령명 또는 내부 규정}

[조건]
- 공문 형식: 수신 / 제목 / 본문(경위-내용-결론) / 붙임
- 어조: 정중하고 명확하게
- 법령 정확성은 담당자가 반드시 재확인
변수 안내{수신처} / {요청 내용} / {답변 방향} / {법령} — 특히 법령명·조항은 실제 확인 필수
추천 도구Claude (조건 준수 정확도 높음)
활용 팁AI가 법령명·조항을 생성하면 반드시 국가법령정보센터(law.go.kr)에서 검증하세요.
프롬프트 1-5
문서 검토 및 개선 제안

작성된 보고서·공문을 제출 전 AI로 품질 점검하고 싶을 때

공공행정 문서 편집 전문가로서 아래 문서를 검토하고 개선점을 제안해줘.

[문서 내용] {작성한 문서 전문 붙여넣기}

[검토 기준]
① 핵심 메시지가 30초 내 파악되는가
② 논리적 흐름이 자연스러운가
③ 공식적·객관적 어조인가
④ 수치·사실 오류가 없는가 (의심 항목 표시)
⑤ 불필요한 중복 표현이 있는가

각 기준별로 평가하고, 수정 제안은 "원문 → 수정안" 형식으로 작성해줘.
변수 안내{작성한 문서} — 개인정보·비공개 정보 입력 금지, 비식별화 후 사용
추천 도구Claude (긴 문서 분석에 강함)
활용 팁내부 기밀·개인정보가 포함된 문서는 절대 AI에 입력하지 말고, 비식별화 후 활용하세요.
SECTOR 02

홍보 · 보도자료

부여군 사업·행사·정책을 알리는 보도자료, SNS 게시물, 군정 소식 콘텐츠를 작성합니다.
프롬프트 2-1
사업·행사 보도자료 작성

신규사업 시작 또는 행사 개최를 언론에 알리는 보도자료가 필요할 때

15년 경력 공공기관 홍보 전문가로서, 충남 지역 기자가 내일 마감 기사에 바로 쓸 수 있는 보도자료를 작성해줘.

[사업·행사명] {사업 또는 행사명}
[일시·장소] {일시} / {장소}
[핵심 메시지 3가지]
- {핵심1}
- {핵심2}
- {핵심3}
[기관장 인용구] {군수 또는 기관장 멘트}

[조건]
- 형식: 제목 / 부제 / 본문 3단락 / 담당자 연락처
- 첫 문장에 가장 강한 메시지 배치
- 어조: 공식적·간결하게
- 1,000자 이내
변수 안내{사업명} / {일시·장소} / {핵심 메시지 3가지} / {기관장 인용구} — 실제 내용으로 교체
추천 도구Claude / ChatGPT
활용 팁"전국 최초", "충남 최초" 등 차별점을 핵심 메시지에 넣으면 기사화 가능성이 높아집니다.
프롬프트 2-2
SNS 홍보 게시물 작성

부여군 공식 SNS(인스타그램·페이스북 등)에 게시할 짧은 홍보 글이 필요할 때

공공기관 SNS 콘텐츠 전문 작성자로서 아래 내용을 SNS 게시물로 변환해줘.

[홍보 내용] {홍보할 사업·행사·정책 내용}
[대상 채널] {인스타그램 / 페이스북 / X(트위터)}
[타깃 독자] {부여군민 / 관광객 / 청년층 등}

[조건]
- 첫 줄에 시선을 끄는 문장 배치
- 핵심 정보 3줄 이내 요약
- 적절한 이모지 포함
- 해시태그 5개 추천
- 150자 이내 (인스타그램 기준)
변수 안내{홍보 내용} / {채널} / {타깃} — 채널에 따라 말투와 길이가 달라집니다
추천 도구ChatGPT (자연스러운 문체·창의성 강점)
활용 팁게시 전 반드시 담당자가 사실 확인 및 최종 표현 검토를 하세요. AI 생성 이모지 남용 주의.
프롬프트 2-3
군정 소식지 기사 작성

부여군 소식지·뉴스레터에 실을 기사나 칼럼 내용이 필요할 때

지역 공공기관 소식지 전문 편집자로서 아래 내용을 소식지 기사 형식으로 작성해줘.

[기사 주제] {주제 또는 부서 사업 내용}
[독자 대상] 부여군민 (전 연령층)
[강조 포인트] {군민 생활과의 연관성, 혜택}

[조건]
- 형식: 제목 / 부제 / 본문 2~3단락 / 문의처
- 어조: 친근하고 쉬운 말투, 전문 용어 최소화
- 분량: 400~600자
- 군민 혜택이 느껴지도록 마지막 문장 마무리
변수 안내{주제} / {강조 포인트} — 군민 생활 연관성을 최대한 구체적으로 제공할수록 좋음
추천 도구ChatGPT / Claude
활용 팁소식지는 전 연령층이 읽으므로 쉬운 언어 사용. "AI 생성"이라는 흔적이 없도록 최종 다듬기 필수.
프롬프트 2-4
보도자료 제목 다듬기

이미 작성한 보도자료의 제목을 더 눈에 띄게 개선하고 싶을 때

신문 헤드라인 작성 전문가로서 아래 보도자료 제목 후보를 평가하고 개선해줘.

[현재 제목] {현재 사용 중인 제목}
[핵심 내용 요약] {보도자료 핵심 2~3줄}

[요청 사항]
① 현재 제목의 장단점 평가
② 기사화 가능성 높은 제목 5가지 대안 제시
③ 각 제목의 강점 한 줄 설명

- 제목은 30자 이내, 핵심 키워드 포함
- "전국 최초", "충남 유일" 등 차별점 부각 가능하면 포함
변수 안내{현재 제목} / {핵심 내용} — 제목 후보가 여러 개라면 모두 입력 가능
추천 도구ChatGPT / Claude
활용 팁최종 제목 선택은 담당자가 판단. 과장되거나 사실과 다른 표현이 포함됐는지 반드시 확인.
SECTOR 03

민원 응대 · 회신

다양한 형태의 민원을 빠르고 정확하게 처리하는 회신문·안내문·분류 업무에 활용합니다.
프롬프트 3-1
민원 회신문 작성

접수된 민원에 대해 공식 회신 문서를 작성해야 할 때

부여군청 민원응대 전문 담당자로서 아래 민원에 대한 공식 회신문을 작성해줘.

[민원 내용 요약] {민원 내용 핵심 정리}
[민원인 정보] {민원인 유형: 주민/사업자/기관 등}
[답변 방향] {긍정적 처리 / 부분 수용 / 불가 및 이유}
[법적 근거] {관련 법령 또는 규정 (있는 경우)}

[형식]
- 공감 표현 → 현재 상태 → 담당 부서 안내 → 처리 일정 → 연락처
- 어조: 정중하고 공식적이되 이해하기 쉽게
- 500자 이내
- 법령은 담당자 재확인 필수 명시
