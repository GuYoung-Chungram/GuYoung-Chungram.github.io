    생각을 정리해준다
    감정, 고민, 계획을 AI와 대화하며 명료하게 정리할 수 있습니다. 24시간 곁에 있는 대화 상대가 생기는 셈입니다.

실습 ChatGPT 맞춤 설정 스마트폰 직접 실습
ChatGPT 맞춤 설정이란?
⚙️

ChatGPT에 "나는 이런 사람이고, 이렇게 대답해줘"를 미리 저장해두면 매번 설명하지 않아도 됩니다. 마치 오랫동안 나를 알아온 비서처럼 행동합니다.

📍 설정 방법 (ChatGPT 앱)
오른쪽 상단 프로필 → 맞춤 설정 → ChatGPT 맞춤 설정 → 두 가지 항목 입력
맞춤 설정 작성 예시 — 직접 따라 입력해보세요
✏️
① ChatGPT에게 나에 대해 알려주기
나에 대한 정보
저는 고양시청에서 22년간 근무하고 곧 퇴직을 앞두고 있습니다. AI는 처음 사용해보는 초보입니다. 건강, 여행, 취미 정보를 자주 찾습니다.
② ChatGPT가 어떻게 대답하면 좋을지
원하는 대답 방식
어렵고 전문적인 단어 없이 쉽게 설명해주세요. 결론을 먼저 말하고, 이유를 설명해주세요. 답변은 짧고 명확하게 해주세요.
✏️
직접 바꿔보세요: 직업, 관심사, 원하는 말투를 나에게 맞게 수정해서 입력하면 훨씬 개인화된 AI 비서가 됩니다.

"좋은 질문"과 "애매한 질문"의 차이 체감
같은 주제, 다른 결과 — 질문을 바꾸면 답이 바뀐다
실습 비교 체험 프롬프트 기초
직접 비교해보세요 — 아래 두 질문을 ChatGPT에 각각 입력해보기
⚡
❌ 애매한 질문
이렇게 물으면?
건강에 좋은 음식 알려줘
😐
너무 일반적인 답이 나옵니다. 내 상황과 맞지 않을 수 있습니다.
✅ 좋은 질문
이렇게 바꾸면?
저는 65세 남성이고 고혈압 약을 복용 중입니다. 혈압 관리에 도움이 되는 식품 5가지를 알려주세요. 각각 왜 좋은지도 간단히 설명해주세요.
✨
나이, 상황, 원하는 형식을 넣으니 훨씬 정확하고 유용한 답이 나옵니다!
좋은 질문의 3가지 재료
🧩
  • 나
    나에 대한 정보
    나이, 상황, 건강상태, 직업 — 내 상황을 알아야 맞춤 답을 줄 수 있습니다
  • 원
    원하는 것을 구체적으로
    "알려줘" 대신 "5가지 알려줘", "표로 만들어줘", "쉽게 설명해줘"처럼 구체적으로 요청합니다
  • 예
    예시나 조건 추가
    "고양시 기준으로", "65세 부부 기준", "예산 3만원 이내"처럼 조건을 주면 훨씬 실용적인 답이 나옵니다

실습 프롬프트 3가지 — 복사 후 ChatGPT에 붙여넣기

1
고양시 보건소 검진 정보 찾기
프롬프트 복사

2
약 복용 중 주의사항 질문하기
프롬프트 복사
저는 고혈압 약(암로디핀)과 고지혈증 약(아토르바스타틴)을 함께 복용 중입니다. 이 약들을 먹을 때 피해야 할 음식이나 행동이 있으면 알려주세요. 쉬운 말로 설명해주세요.
⚠️
주의: AI 답변은 참고용입니다. 실제 복약 여부는 담당 의사나 약사에게 반드시 확인하세요.

Canvas란? — 1분 개념
🖼️
  • 1
    ChatGPT Canvas
    채팅 옆에 편집 창이 열림 → AI가 글·코드를 바로 작성·수정해줌. 글쓰기·HTML 제작에 최적.
  • 2
    Gemini Canvas (아티팩트)
    대화 옆 오른쪽 창에 미리보기 화면이 뜸 → HTML을 만들면 바로 화면에서 동작하는 모습 확인 가능.
  • 3
    HTML이란?
    웹페이지를 만드는 언어. 컴퓨터에 저장하면 앱처럼 스마트폰에서 열림. 인터넷 없어도 동작.
STEP 1 — 내 복약 정보를 AI에게 알려주기
💬
💡
먼저 이렇게 시작합니다 — ChatGPT 또는 Gemini를 열고, 아래 프롬프트를 복사해서 붙여넣으세요.
기본 프롬프트 — 복사 후 내 약 이름으로 바꾸세요
나는 다음 약을 매일 복용합니다. - 암로디핀 5mg: 아침 식후 - 아토르바스타틴 10mg: 저녁 식후 - 아스피린 100mg: 아침 식전 (위장약과 함께) 이 복약 정보를 보기 좋은 표로 만들어줘. 각 약마다 복용 시간, 주의사항(술·자몽 금지 등)을 포함해서 보기 쉽게 글씨 크게, 색깔 구분해서 만들어줘.
✏️
약 이름은 내 것으로 바꾸세요! 암로디핀, 아스피린 대신 실제 복용 중인 약 이름을 넣으면 됩니다. 모르면 약봉투에서 확인!
STEP 2 — 앱으로 만들어달라고 요청하기 (핵심!)
🛠️
⭐
이게 핵심입니다! STEP 1에서 표를 받은 뒤, 바로 이어서 아래 프롬프트를 보내세요. AI가 실제로 동작하는 HTML 앱을 만들어줍니다.
A
ChatGPT Canvas용 프롬프트
ChatGPT에 붙여넣기
위 복약 정보를 스마트폰에서 쓸 수 있는 HTML 페이지로 만들어줘. 기능: 1. 아침/점심/저녁으로 구분된 복약 카드 2. 각 칸마다 체크박스 (오늘 먹었는지 표시) 3. 글씨는 크게, 배경은 흰색, 색깔로 시간대 구분 4. 맨 위에 오늘 날짜 자동 표시 5. 인쇄 버튼 Canvas에서 HTML로 만들어줘.
G
Gemini Canvas(아티팩트)용 프롬프트
Gemini에 붙여넣기
위 복약 정보로 스마트폰 친화적인 HTML 복약 도우미를 만들어줘. 조건: - 아침/저녁 시간대 카드형 레이아웃 - 각 약 이름, 용량, 주의사항 표시 - 복용 체크박스 (클릭하면 색 바뀜) - 오늘 날짜 표시 - 쓰기 편하게: 버튼 크게, 글씨 크게 오른쪽 미리보기 창에서 바로 확인할 수 있게 만들어줘.
+
추가 요청 — 마음에 안 들면 이렇게 수정 요청
글씨 더 크게
글씨를 더 크게 해줘. 최소 18px 이상으로.
약 추가
메트포르민 500mg 저녁 식후 약을 추가해줘.
색상 변경
아침은 노란색, 저녁은 파란색으로 카드 색을 바꿔줘.
메모 기능 추가
각 약마다 오늘 특이사항 메모를 입력할 수 있는 칸을 추가해줘.
STEP 3 — 저장하고 스마트폰에서 쓰기
📱
1
HTML 코드 복사
Canvas 창에서 만들어진 코드를 전체 선택(Ctrl+A) → 복사(Ctrl+C)
2
메모장에 붙여넣기 → 파일로 저장
윈도우 메모장 열기 → 붙여넣기 → 파일명: 나의복약표.html 로 저장
⚠️ 저장할 때 '모든 파일'로 형식 변경 필수
3
스마트폰으로 전송
카카오톡 나에게 보내기 → 파일 첨부 → 스마트폰에서 다운로드 → 크롬으로 열기
4
홈화면에 저장 (선택)
크롬에서 열린 후 → 우상단 ⋮ 메뉴 → 홈 화면에 추가 → 앱처럼 사용 가능!
💡 가족 공유 팁
만든 HTML 파일을 카카오톡으로 가족에게 전송하면 부모님, 배우자도 같은 복약표를 쓸 수 있습니다.
한 번만 만들면 인터넷 없이도 스마트폰에서 열립니다.
★ 더 나아가기 — 알림 메시지 자동 생성
🔔
앱을 만든 뒤, AI에게 카카오톡 알림 메시지도 만들어달라고 요청할 수 있습니다.
알림 메시지 생성 프롬프트
위에서 만든 복약 정보를 바탕으로 매일 아침 8시, 저녁 8시에 보낼 카카오톡 알림 메시지를 만들어줘. 조건: - 따뜻하고 다정한 말투 - 이모지 포함 - 어떤 약을 먹어야 하는지 명확하게 - 2줄 이내로 짧게 아침용과 저녁용 각각 3가지씩 만들어줘.

모듈3 · 실습 08 — 더 잘 쓰는 법
잘 물어보는 법 3가지
역할 부여 · 구체적 요청 · 예시 제공 — 프롬프트 3원칙
강의+데모 프롬프트 심화 역할·구체·예시
프롬프트 3원칙 — 이것만 알면 AI 대화가 달라집니다
🎯
  • 역
    역할 부여 — "너는 ○○○이야"
    AI에게 전문가 역할을 주면 그 분야 전문가처럼 답합니다.
    예) "너는 20년 경력의 건강 영양사야. 내 식단을 점검해줘."
  • 구
    구체적 요청 — 숫자, 형식, 조건 명시
    "알려줘" 대신 "5가지로", "표로", "한 페이지로", "초등학생도 이해하게" 등 구체적으로 요청합니다.
    예) "3가지만", "단계별로", "장단점 비교표로"
  • 예
    예시 제공 — "이런 식으로 해줘"
    원하는 결과의 예시를 보여주면 AI가 그 스타일을 따라합니다.
    예) "이렇게 써줘: '안녕하세요, 저는 ○○입니다. 오늘 말씀드릴 내용은...'"
3원칙 조합 실습 — 퇴직예정자 맞춤 예시
⚡
역할+구체+예시 모두 담은 프롬프트
// 역할 부여 너는 퇴직 후 재취업을 전문으로 돕는 커리어 코치야. // 내 상황 (구체적) 나는 고양시청에서 22년간 총무 행정을 담당하다 퇴직하는 62세야. 앞으로 중소기업 총무 파트타임이나 컨설팅을 하고 싶어. // 요청 (구체적) 이력서 자기소개서 첫 줄을 5가지 버전으로 써줘. // 예시 제공 예시처럼: "22년 공공기관 총무 행정 전문가, 현장에서 검증된 실행력을 제공합니다."
✨
위 3가지 원칙을 다 담지 않아도 됩니다. 하나씩 추가할 때마다 답변이 얼마나 달라지는지 직접 비교해보세요!

강의+데모 Gemini Gems 멘탈코치 이메일봇
Gems란? — 나만의 AI 비서를 저장해두는 기능
💎

Gemini Gems는 내가 원하는 역할과 방식을 AI에게 미리 저장해두는 기능입니다. 한 번 만들어두면 언제든지 같은 AI 비서를 다시 불러쓸 수 있습니다.

✅ Gems의 장점
• 무료로 만들고 사용 가능
• 매번 긴 설명 반복 불필요
• 나만의 AI 비서가 생김
• 구글 계정만 있으면 OK
📍 만드는 방법
① gemini.google.com 접속
② 왼쪽 메뉴 → Gems 탐색
③ 새 Gem 만들기 클릭
④ 이름 + 지침 입력 → 저장
Gem 1 — 멘탈코치 (강사 시연 후 직접 만들기)
🧠
💎 자기인식 리더십 챗봇
퇴직 후 감정을 함께 들여다보고, 작게 다시 움직일 수 있도록 돕는 AI 인생코치
Gems 지침 — 복사 후 붙여넣기
# 역할 당신은 사용자의 감정에 공감하고, 감정 뒤 동기와 패턴을 탐색하며, 작고 실현 가능한 행동으로 다시 움직이도록 돕는 코칭형 AI 인생코치입니다. # 대화 시작 — 접속 시 반드시 이 인사말로 시작 안녕하세요 😊 저는 당신의 감정을 함께 들여다보고, 다시 작게 움직일 수 있도록 돕는 AI 인생코치입니다. 이곳은 안전한 대화 공간이에요. 마음 편히 이야기해 주세요. 아래 번호 중 하나를 입력해 주세요. 1 — 요즘 자꾸 미루게 되는 일이 있어요 2 — 하루가 텅 빈 느낌이에요 3 — 마음이 크게 흔들린 순간이 있었어요 4 — 오늘 감사한 마음이 올라왔어요 5 — 뭔가를 시작하려는데 두려움이 앞서요 6 — 오랫동안 해온 일을 놓아야 한다는 게 실감이 안 나요 7 — 퇴직 후 내가 누구인지 솔직히 잘 모르겠어요
