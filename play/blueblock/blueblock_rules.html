120개의 블록 카드 중에서 규칙에 맞는 블록을 골라
A · B · C 세 줄을 순서대로 완성하는 팀 게임입니다.
블록이란?
모든 블록은 알파벳 1글자 + 숫자 4자리로 이루어져 있어요.
A3459
A 블록
B5731
B 블록
C7883
C 블록
알파벳은 A, B, C 세 종류만 있어요. 블록 코드를 직접 입력해서 게임에 참여합니다.
세트 만드는 법
1세트 = A블록 1개 + B블록 1개 + C블록 1개
단, 아래 두 가지 조건을 모두 만족해야 해요!
조건 ① 가로 — A 숫자 < B 숫자 < C 숫자
같은 세트 안에서, A의 숫자보다 B의 숫자가 더 커야 하고, B의 숫자보다 C의 숫자가 더 커야 해요.
조건 ② 세로 — A끼리, B끼리, C끼리도 오름차순
1번 세트의 A보다 2번 세트의 A 숫자가 더 커야 해요. B와 C도 마찬가지예요. 즉, 위에서 아래로 숫자가 점점 커져야 합니다.

불량 블록 제거
120개 블록 중 일부는 불량 블록이에요. 불량 블록은 세트에 사용할 수 없어요!
아래 두 가지 규칙 중 하나라도 해당되면 불량이에요.
불량 규칙 ①
숫자 중간에 '00'이 연속으로 있으면 불량!
숫자 4자리의 가운데 두 자리가 00이면 불량이에요.
단, 맨 앞이나 맨 뒤가 00인 건 괜찮아요!
A3002
3·00·2 → 불량❌
B1003
1·00·3 → 불량❌
A0038
앞00 → 양호✅
C9000
뒤00 → 양호✅
불량 규칙 ②
숫자 4개를 더했을 때 10, 20, 30이 되면 불량!
블록 코드의 숫자 4자리를 모두 더해서 10, 20, 30이 되면 불량이에요.
A1234
1+2+3+4=10 ❌
B3854
3+8+5+4=20 ❌
A1235
1+2+3+5=11 ✅
